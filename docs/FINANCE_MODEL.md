### Формулы
```
GI (валовый доход) = сумма(дней × загрузка × ADR) по месяцам
Чистый доход = GI × 55%
Налог = Чистый доход × 6%
NOI = Чистый доход - Налог = GI × 55% × 94%
ROI = NOI / Стоимость × 100%
Цена капитализации = NOI / Cap Rate
```

### Пример: 1-комн 41.4 м² за 20 658 600 ₽
- GI год 1: 6 211 200 ₽
- NOI год 1: 3 211 190 ₽
- ROI год 1: 15.5%
